YouTube Takeout

  1. Go to takeout.google.com
  2. Select "YouTube and YouTube Music"
  3. Check "history" → Export as JSON
  4. Download and upload the JSON file here

TikTok / Douyin

  1. TikTok: Settings → Privacy → Download your data → JSON
  2. Douyin: Profile → Menu → Privacy → Download data
  3. Upload the JSON file, or paste titles in Paste tab

Bilibili

  1. Open Bilibili → History page
  2. Copy video titles
  3. Paste in the Paste Text tab
