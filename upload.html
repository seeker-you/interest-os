📤 导入观看记录

支持 YouTube、B站、TikTok、抖音等所有平台。文件上传或直接粘贴视频标题列表均可

☁️
拖拽文件到此处
支持 .json .csv .txt（任意平台导出） · 最多 200 条记录

▶ YouTube 导出方法

  1. 访问 Google Takeout
  2. 选择「YouTube 和 YouTube Music」
  3. 仅勾选「历史记录」→ 导出 JSON 格式
  4. 下载后上传到本页面

▶ B站导出方法

  1. 打开 B站 → 历史记录页面
  2. 使用浏览器插件导出或逐页复制标题
  3. 粘贴到「粘贴文本」标签页

▶ TikTok / 抖音导出方法

  1. TikTok：设置 → 隐私 → 下载你的数据 → 选择 JSON 格式
  2. 抖音：我 → 右上角三条线 → 隐私设置 → 个人信息下载
  3. 下载后上传 JSON 文件，或直接复制标题到「粘贴文本」

▶ 通用方法

直接复制你的观看/收藏夹中的视频标题，粘贴到本页面即可。每行一个标题。
