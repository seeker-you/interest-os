YouTube Takeout

  1. Visit takeout.google.com
  2. Select "YouTube and YouTube Music"
  3. Check "history" → Export as JSON
  4. Download and upload here

TikTok / Douyin

  1. TikTok: Settings → Privacy → Download your data → JSON
  2. Douyin: Profile → Menu → Privacy → Download data
  3. Upload JSON file, or paste titles in "Paste Text" tab

Bilibili

Go to History page, copy titles manually, paste in the "Paste Text" tab.
